Attribute与Property

发表于 2017-06-15 | 分类于 手记

Property

每一个DOM节点都是一个HTMLElement对象，像其它js对象一样，可以添加一些自定义的方法和属性。这些方法和属性就是Property。它只能被js读取，且不会影响HTML的渲染。

Attribute

与Property不同，Attribute会在DOM节点上显示出来，但不可以在DOM对象中通过for-in遍历出来。

有什么区别？

Property Attribute
值可以任意类型 值只能是字符串
键名区分大小写 键名不区分大小写
在innerHTML里面不可见 在innerHTML里面可见
